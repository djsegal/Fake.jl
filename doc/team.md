# Fake.Team
```julia
# Random Team Creature
Fake.Team.creature() # => "gooses"

# Random Team Name created from random US State (Fake.Address.state) prepended to a random Team Creature
Fake.Team.name() # => "Oregon Vixens"

# Random Team State
Fake.Team.state() # => "Oregon"

# Random Team Sport
Fake.Team.sport() # => "lacrosse"

# Random Team Mascot
Fake.Team.mascot() # => "Hugo"
```
